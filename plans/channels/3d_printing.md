# 3D Printing (3dp.maiks-yt.com)

## Vision
A visual gallery and status portal for 3D printing and design.

## Core Features
- **3D Model Viewer**: A web-based STL/3MF viewer for viewing 3D models.
- **Print Queue Status**: Real-time status updates for the 3D printer (temperature, progress, etc.).
- **Time-Lapse Gallery**: A collection of time-lapse videos showing prints from start to finish.
- **Filament Inventory**: A tracker of filament usage and current stock levels.

## Technical Goals
- Layered texture UI with geometric patterns.
- Integration with popular slicers and printer management software.
- AI-assisted print failure detection.
