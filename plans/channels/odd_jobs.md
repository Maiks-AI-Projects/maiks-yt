# Odd Jobs (oj.maiks-yt.com)

## Vision
A rugged, workshop-inspired space for miscellaneous tasks and variety content.

## Core Features
- **Project Progress Tracker**: Real-time status updates for ongoing projects.
- **Workshop Inventory**: A list of common tools and materials in the workshop.
- **Task List**: A community-voted task list for variety content.
- **Collaborative Projects**: A space for viewers to contribute and collaborate on projects.

## Technical Goals
- Workshop-inspired UI with "under construction" motifs.
- Integration with popular project management tools.
- AI-assisted task prioritization and project estimation.
