# Tech (tech.maiks-yt.com)

## Vision
A comprehensive hub for hardware reviews, teardowns, and general tech news.

## Core Features
- **Hardware Teardown Gallery**: Detailed photos and diagrams of hardware teardowns.
- **Tech News Feed**: Aggregated updates from the latest tech publications.
- **Review Registry**: A collection of the creator's tech reviews.
- **Device Comparisons**: Interactive tools for comparing hardware specs.

## Technical Goals
- Minimalist, brushed-metal UI.
- Integration with major tech news APIs.
- AI-assisted tech specification lookup.
