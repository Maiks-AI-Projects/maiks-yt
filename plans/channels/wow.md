# World of Warcraft (wow.maiks-yt.com)

## Vision
An epic community hub for Azeroth's finest, focusing on raid progression and character sharing.

## Core Features
- **Guild Progress Tracker**: A live dashboard showing current progress in raids and dungeons.
- **Talent Build Sharing**: A tool for sharing and importing talent builds for different classes.
- **Character Profile Integration**: Linking character data from the Battle.net API.
- **Raid Schedule and Signup**: A centralized calendar for community-run raids and signups.

## Technical Goals
- High-performance, low-latency API integration with Battle.net.
- Ornate, dark-themed UI with epic, gold/purple accents.
- Custom skins that match the WoW aesthetic.
