# Custom Donation System

## Transparency Vision
The system provides full transparency into the project's finances.
- **Income/Expense Tracking**: All incoming donations and outgoing project costs are logged and displayed publicly.
- **Budget Categories**: Donations can be allocated to specific budgets:
  - **Tech Spending**: Hardware, server costs, software licenses.
  - **Grocery/Clothing**: Supporting the creator's daily life, clearly separated from project funds.
  - **Project Specific**: Direct funding for a particular goal (e.g., a specific AI model or game server).

## Verification of Donation Usage
Transparency isn't just about logs; it's about proof.
- **Proof of Purchase**: Upload receipts or screenshots for significant project-related expenses.
- **Usage Verification**: AI-assisted verification of hardware usage and project milestones to ensure funds are spent as intended.

## Hardware Transparency
Each hardware-related pot will include a detailed parts list with:
- Part Name
- Store Link (e.g., Amazon, Mindfactory)
- Current Price
- Progress towards goal

## Public Ledger
A transparent, real-time list of all donations (with privacy options for donors).
- Date and Time
- Donor Name (or Anonymous)
- Amount
- Targeted Pot/Budget
- Message
