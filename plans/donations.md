# Custom Donation System

## Donation 'Pots'
Donations can be directed towards specific goals or a general fund.
- **General Fund**: Used for site maintenance, hosting, and general improvements.
- **Specific Pots**: Hardware upgrades (e.g., "New GPU Pot"), specific game servers, or community events.

## Hardware Transparency
Each hardware-related pot will include a detailed parts list with:
- Part Name
- Store Link (e.g., Amazon, Mindfactory)
- Current Price
- Progress towards goal

## Public Ledger
A transparent, real-time list of all donations (with privacy options for donors).
- Date and Time
- Donor Name (or Anonymous)
- Amount
- Targeted Pot
- Message
