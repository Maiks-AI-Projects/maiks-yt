# Stream Overlays and Notifications

## Overlay Design
- **Animated Sliding Cards**: Used for showing current goal progress, latest donation, and recent followers.
- **Real-time Notifications**: Instant visual feedback for engagement and donations.
- **Responsive Layout**: Overlays adjust based on the stream scene (Gaming, Just Chatting, Brb).

## Integration
- **Combined Chat**: A unified chat overlay that integrates both YouTube and Twitch chat into a single, cohesive feed.
- **Engagement Triggers**: Streamers can trigger specific animations or sounds via a dashboard.

## Technical Details
- Built with HTML/CSS/JS for easy customization and OBS integration (Browser Source).
- WebSockets for real-time data updates.
