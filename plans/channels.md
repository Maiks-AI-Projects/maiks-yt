# Channel Vision

## Subdomain-Specific Hobbies
- **mc.maiks-yt.com**: Minecraft content, server status, and player stats.
- **ht.maiks-yt.com**: HyTale updates, community news, and gameplay highlights.
- **sf.maiks-yt.com**: Satisfactory factory builds, efficiency guides, and save file sharing.
- **talking.maiks-yt.com**: General discussions, podcasts, and community hangouts.
- **wow.maiks-yt.com**: World of Warcraft raids, dungeon runs, and character progression.
- **me.maiks-yt.com**: Micro Electronics projects, circuit design, and soldering.
- **3dp.maiks-yt.com**: 3D Printing designs, printer tuning, and time-lapses.
- **prog.maiks-yt.com**: Programming tutorials, code reviews, and open-source contributions.
- **bd.maiks-yt.com**: Brain Damaged (Comedy/Chaos) - High-energy, unpredictable content.
- **tech.maiks-yt.com**: General tech reviews, hardware teardowns, and news.
- **ai.maiks-yt.com**: AI research, model training, and generative art.
- **out.maiks-yt.com**: Outdoors adventures, hiking, and travel vlogs.
- **oj.maiks-yt.com**: Odd Jobs - Variety content covering miscellaneous tasks and projects.

## Unique Skinning per Subdomain
Each subdomain will have a unique theme (CSS/Tailwind) that matches the aesthetic of the hobby.
- **mc**: Blocky, green/brown color palette, pixelated fonts.
- **ht**: Stylized, vibrant colors, fantasy-inspired UI elements.
- **sf**: Industrial, sleek, orange/gray color palette, high-tech UI.
- **talking**: Clean, modern, focus on readability and community interaction.
- **wow**: Epic, dark, gold/purple accents, ornate UI frames.
- **me**: Schematic-style lines, neon accents, dark backgrounds.
- **3dp**: Layered textures, geometric patterns, "print-in-progress" visuals.
- **prog**: Code editor aesthetic, monospaced fonts, syntax highlighting colors.
- **bd**: Glitch effects, high-contrast, chaotic but functional layout.
- **tech**: Minimalist, high-tech, brushed metal and glass effects.
- **ai**: Ethereal, neural network patterns, glowing gradients.
- **out**: Natural tones (earthy greens/blues), organic shapes.
- **oj**: Workshop aesthetic, "under construction" motifs, rugged UI.

## Hobby-Specific Features
- **General Functions**:
  - **Server Access**: Integrated server status and direct connection links.
  - **Blueprint Sharing**: Dedicated repository for game blueprints, code snippets, or 3D models.
  - **AI Verification**: Automated content tagging and verification using AI models.
- **mc**: Live server map integration and player inventory viewer.
- **ht**: Character profile cards and guild/clan listings.
- **sf**: Factory production calculators and interactive build maps.
- **wow**: Guild progress tracker and talent build sharing.
- **me**: Interactive circuit diagrams and component inventory.
- **3dp**: 3D model viewer (STL/3MF) and print queue status.
- **prog**: Live code repository integration and interactive REPLs.
- **ai**: Model playground and generated gallery.
