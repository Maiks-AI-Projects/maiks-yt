# Viewer Profiles and Engagement

## Features
- **Verified Game Characters**: Users can link their game accounts (Minecraft, HyTale, WoW) to their profile.
- **Badges**: Special badges for "Verified Player", "Top Donor", "Beta Tester", etc.
- **Dynamic Profile Pictures**: The viewer's profile picture can dynamically change based on the game they are currently playing or the most active game on the channel.

## Gamification
- **Leveling System**: Viewers earn experience points (XP) for watching, chatting, and donating.
- **Inventory**: Collectible items (e.g., virtual stickers, emotes) earned through engagement.
