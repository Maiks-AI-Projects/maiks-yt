# Stream Website and Overlay Idea Cards

This folder collects rough product ideas for the stream website and overlay project.

The cards are intentionally small. Each one should capture:

- what the idea is
- why it might be useful
- what data or integrations it needs
- what would be necessary to build it
- open questions before committing to it

Nothing here is final architecture yet. These cards are meant to help compare ideas later and decide whether this should become one project, multiple sites, or a shared platform with separate public surfaces.
