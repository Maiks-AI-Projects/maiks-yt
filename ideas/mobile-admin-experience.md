# Mobile Admin Experience

## Idea

Some stream or site emergencies may need to be handled from a phone.

This does not mean every admin screen needs to be mobile-first, but critical actions should work on mobile.

## Why It Matters

If the streamer is away from the main PC, dealing with OBS, or needs to respond quickly, a phone-friendly admin flow can help.

Possible mobile actions:

- cancel a scheduled stream
- trigger emergency clean mode
- hide chat overlay
- mute AI
- approve or reject an urgent action
- post a quick update
- check whether overlays are connected

## Build Requirements

- responsive critical admin pages
- mobile-safe buttons for urgent actions
- confirmation for destructive/risky actions
- compact action panel view
- mobile schedule cancellation flow
